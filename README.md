# 📱 PokéSorteador Mobile - React Native

Uma aplicação mobile desenvolvida em **React Native** com **Expo** que consome a [PokeAPI](https://pokeapi.co/) para sortear Pokémons aleatórios de forma dinâmica. Este projeto representa a evolução de uma aplicação web legada para o ecossistema mobile, aplicando conceitos de componentização, estilização nativa e gerenciamento de estados assíncronos.

---

## 🚀 Funcionalidades

* 🎲 **Sorteio Aleatório:** Gera um ID dinâmico entre 1 e 1025 para buscar qualquer Pokémon da base de dados oficial.
* 🖼️ **Renderização de Mídia:** Consome e exibe a arte oficial (`official-artwork`) em alta definição de cada criatura.
* 📊 **Mapeamento de Dados:** Trata as informações da API para exibir o nome formatado em caixa alta e a listagem de tipos unidos por barra (ex: `GRASS / POISON`).
* ⏳ **Feedback Visual (UX):** Exibe um indicador de carregamento nativo (`ActivityIndicator`) enquanto as requisições assíncronas são processadas, evitando telas travadas para o usuário.

---

## 🛠️ Tecnologias Utilizadas

* **React Native** (v0.76)
* **Expo** (SDK 54)
* **JavaScript (ES6+)** com Async/Await e Fetch API
* **Estilização Nativa:** StyleSheet (Layout baseado em Flexbox)
